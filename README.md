# Minutes

A fast, browser-based form for completing **Elementary ARD Deliberations** meeting minutes and printing them to PDF — no software installation required.

## Usage

1. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
2. **Fill in the blanks** — click any underlined field and type.
3. **Bubble in the dots** — click the ○ circles next to completed items to fill them (●).
4. Click the green **🖨 Generate PDF** button (top-right) and choose **Save as PDF** from your browser's print dialog.

Your entries are automatically saved in your browser's local storage so you can close and reopen the form without losing work.  
Click **Clear Form** (red button) to reset everything and start fresh.

## Fields

| Field | Description |
|---|---|
| Student Name | Full name (Last, First Middle) |
| Campus | School name |
| Grade | Current grade level |
| Date | Meeting date |
| ARD Notice Sent Date | Date notice was sent home |
| Purpose of Meeting | Reason(s) for the meeting |
| Parent Concerns | Concerns expressed by the parent |
| Eligibility | Current disability eligibility(s) |
| IEP Goal Areas | Areas where goals/objectives were accepted |
| Personal Care Services | Areas where PCS are provided |
| LRE Placement | Placement of services description |
| ESY Sentence | Extended School Year determination |
| Student First Name | Used in the committee decision summary |
| Therapy Sentence | Describes any related therapy services |
| Closing Comments | Additional closing notes |
| Minutes Taker | Name and position of the person taking minutes |

Bubble checkboxes are provided for every statement that requires committee confirmation.
